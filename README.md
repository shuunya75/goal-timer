# goal-timer

目標時間・最低目標・経過時間・残り時間を同時に表示する、筋トレやプランク向けのシンプルなタイマーです。

## 公開URL

GitHub Pagesで公開しています。

https://shuunya75.github.io/goal-timer/

## ローカルで開く方法

このディレクトリで以下を実行します。

```bash
python3 -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000
```

## GitHub Pages

このサイトは静的なHTML / CSS / JavaScriptアプリとして、GitHub Pagesの `main` ブランチ `/` から公開する想定です。
